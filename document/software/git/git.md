### git 기본

#### 1.1. git 명령어

1. git clone
```bash
]$ git clone {url}
```
2. git status
```bash
]$ git status
```

3. git add
```bash
]$ git add *
```
(or)
```bash
]$ git add {파일명}
```

4. git commit
```bash
]$ git commit -m "{comment}"
```

5. git push
```bash
]$ git push origin master
```
